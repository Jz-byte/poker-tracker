Open Google Apps Script

Go to script.google.com and click + New project (top-left).

2

Paste the backend code

Delete everything in the code editor, then paste the contents of poker-backend.js (in the same folder as this file).

Save the project — name it Poker Tracker.

3

Deploy as a Web App

Click Deploy → New deployment.
Click the ⚙ gear icon next to Type → select Web app.
Set Execute as: Me
Set Who has access: Anyone
Click Deploy → authorize your Google account → copy the URL.

4

Paste your Web App URL here

The URL looks like: https://script.google.com/macros/s/AKfyc…/exec

Troubleshooting checklist:
1. Make sure you replaced all the Apps Script code with the new poker-backend.js
2. After updating the code, go to Deploy → Manage deployments → ✏️ Edit → New version → Deploy
3. Use the same URL — it doesn't change when you redeploy
4. You can verify the script works by clicking here to test it directly in a new tab — you should see {"ok":true,...}
